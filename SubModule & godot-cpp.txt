Scaricare godot-cpp non come submodulo (in un folder proprio)

	git clone https://github.com/godotengine/godot-cpp.git
	cd godot-cpp
	git checkout godot-4.0.2-stable oppure git checkout godot-3.5.2-stable
	git submodule update --init --recursive

Aggiungere il progetto godot-cpp utilizzando il branch corretto (sostituire 3.x con il tag del branch che deve conicidere con la versione dell'engine che si utilizza)

	git submodule add -b 4.0 https://github.com/godotengine/godot-cpp oppure git submodule add -b 3.x https://github.com/godotengine/godot-cpp
	cd godot-cpp
	git submodule update --init
	
Modificare una versione di godot-cpp

	editare il file .gitmodules impostando la versione
	git submodule update --init --recursive --remote
	cd godot-cpp
	git submodule update --init
	git submodule status
	
	cd godot-cpp
	git checkout master
	git pull
	git checkout <tag-name>	(es: git checkout godot-3.4.2-stable)

Eliminare e reinserire godot-cpp

	git rm -f godot-cpp
	git commit -m wip
	git clone https://github.com/godotengine/godot-cpp.git
	git submodule add https://github.com/godotengine/godot-cpp	(git submodule add -b 4.0 https://github.com/godotengine/godot-cpp oppure git submodule add -b 3.5 https://github.com/godotengine/godot-cpp)
	git checkout godot-4.0.2-stable oppure git checkout godot-3.5.2-stable
	git submodule update --init --recursive
